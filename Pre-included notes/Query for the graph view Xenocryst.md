Thing you can copy paste in query field of your graph view so it's all cool like mine.

 ```
-path:".base" -path:"Path/To/Your/Daily Notes" -file:"Untitled" -["query"] -file:"Xenocryst config v2" -path:"Xenocryst bundle/Templates"
 ```

You can replace `-path:"Path/To/Your/Daily Notes"` by the actual path to the folder where you put your daily notes.
