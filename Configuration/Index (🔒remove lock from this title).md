
> [!info] What's the Index ?
> This note is meant to hold a list of every single one of your files, stored down below as a bullet point list of markdown links.
> 
> The "shelved" checkbox feature needs this list to be able to find all Moment notes inside that list, filter down only the ones that occured during the relevant time period, and then check all of them for shelved properties.
> 
> This feature is purely optional. You don't need it. Just like you don't need every other part of Xenocrysts.
> 
> I've included a script that can automatically generate this list. It's in JavaScript, the native language of Obsidian, which means it will run very fast, even on mobile. You can run it via the [CustomJS](obsidian://show-plugin?id=customjs) plugin.
>  
> After installing the plugin. Add `Xenocryst bundle/Templates` to the list of folders in the CustomJS settings, and then, the script should work automatically from then on, and update the index everytime it detects a new file has been created.
> 
> The index should appear below this very code block. Try it.
